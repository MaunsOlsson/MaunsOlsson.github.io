Webscraping reviews on Metacritic and creating an interactive analysis in Power BI

To interact with the Power BI Report it is recommended to view the report in fullscreen. This can be done by pressing the enlarge button on the button right. You can also download the report either directly from the shown report on the github button or on the websites github icon.

Note: R is only supported in Power BI Desktop and can not run code in Power BI service - which might create a problem with some visualisations. It is hence recommended, if there is errors on page to download the report on your own computer. Link is in the dashboard.

From first page on BI-report:

This project was initiated to leverage skills in web scraping, R, HTML code and Power BI to examine and demonstrate the potential of using scientific programming in reports or data analysis. The primary objectives were:
  • Extract larger amount of information from a website with multiple layers of pages.
  • Design a dashboard.
  • Create statistical analysis that can interact with the user.
  • Examine how R can be implemented in Power BI
To achive this for my project it required an understanding of:
  1. Data cleaning and transformation
  2. Power Querry
  3. Dashboard design
  4. Parameters, buttons and bookmarks
  5. R coding
  6. HTML code
  7. CSS and XML selectors
  8. Webscraping through Rvest
  9. Visualisation through both R and Power BI
  10. DAX coding
  11. Regressions and other statistical models
